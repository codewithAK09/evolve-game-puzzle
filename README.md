# EVOLVE — Game Puzzle

Minimal static HTML/CSS/JS puzzle game with animated PCB-style background and cursor sparkles.

Repository: https://github.com/codewithAK09/evolve-game-puzzle.git

## Preview
Open `index.html` in a browser or run a simple static server to try locally.

## Run locally
From the project root:

```bash
python3 -m http.server 8000
# then open http://localhost:8000
```

## Notes
- Static site — no build step required.
- Designed for modern desktop browsers. For best effect, use Chrome/Firefox.
